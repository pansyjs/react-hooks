# useFavicon

设置页面的 favicon hook。

## 代码演示

<code src="./demo/demo01.tsx"></code>

## API

```ts
useFavicon(href: string)
```

### Params

| 参数 | 说明                                                 | 类型     | 默认值 |
|------|------------------------------------------------------|----------|--------|
| href | favicon地址, 支持 `svg`/`png`/`ico`/`gif` 后缀的图片 | `string` | -      |
